Usage

Use the cURL to publish a message


                  curl -X POST {{ .SdkEndpoint }}/message \
                    -H "Content-Type: application/json" \
                    -H "Idempotency-Key: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx" \
                    -H "Authorization: {{ .Auth.Authorization }}" \
                    -d '{
                      "app_id": "{{ .App.Id }}",
                      "type": "{{ .MessageType }}",
                      "body": "{\"hello\":\"world\",\"timestamp\":\"{{ .Timestamp }}\"}",
                      "headers": { "x-client": "kanthor/{{ .Sdk.Version }} playground" }
                    }'
